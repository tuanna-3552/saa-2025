---
description: Review recent changes and wrap up the current work session
---

Review the current branch and the most recent commits.

1. Run `git status` and `git log --oneline -10`.
2. Summarize what was modified, added, or removed.
3. Analyze the overall impact and quality of the changes.
4. List any uncommitted files that still need attention.
5. Flag anything that looks risky or incomplete.

**IMPORTANT**: Do NOT start implementing. This is a review-only command.
